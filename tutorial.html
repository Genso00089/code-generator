  1. 「FormBridge」を選択
  2. 「Latest (最新版)」を選択
  3. 「郵便番号→住所変換」にチェック
  4. 下にスクロールして「✨ コードを生成」をクリック
5 フィールド名の設定

FormBridgeで使用しているフィールド名を正確に入力してください。

⚠️ 重要： フィールド名は、FormBridgeの設定画面で確認できる「フィールドコード」と完全に一致させる必要があります。

よく使用されるフィールド名の例：

郵便番号フィールド: 郵便番号, zipcode, postal_code 住所フィールド: 住所, address, 住所1 氏名フィールド: 氏名, name, お名前 電話番号フィールド: 電話番号, tel, phone 日付フィールド: 予約日, date, 希望日 生年月日フィールド: 生年月日, birthday, birth_date
6 カスタム機能の依頼（上級者向け）

既存の機能にない独自の処理が必要な場合は、カスタム機能を依頼できます。

📝 カスタム機能の例

機能名： 電話番号フォーマット機能

やりたいこと： 電話番号を入力時に自動的にハイフンを入れて整形したい

実装方法： 入力中にリアルタイムでフォーマットを適用

対象フィールド： 電話番号, 携帯電話

💡 ヒント： カスタム機能を使用する場合は、「カスタム機能」にチェックを入れてから詳細を記入してください。
7 コードの生成と使用

「✨ コードを生成」ボタンをクリックすると、JavaScriptコードが自動生成されます。

📋 生成されたコードの使用方法
  1. 「📋 コピー」ボタンでコードをコピー
  2. FormBridgeの管理画面を開く
  3. 「JavaScript / CSS でカスタマイズ」を選択
  4. コピーしたコードを貼り付け
  5. 「保存」をクリック

生成されるコードの例：

// 郵便番号から住所を自動取得 (FormBridge v2 API) formBridge.events.on('form.field.change.郵便番号', async (context) => { const postalCode = context.value; if (postalCode && typeof postalCode === 'string') { const address = await handlePostalCodeLookup(postalCode); if (address) { context.setFieldValue('住所', address); } } });
8 動作確認とトラブルシューティング

コードを適用した後は、必ず動作確認を行いましょう。

✅ 確認項目

  • フィールドに値を入力したときに期待通りに動作するか
  • エラーメッセージが表示されていないか
  • 他の機能に影響を与えていないか

🔧 よくある問題と解決方法

問題： コードが動作しない
解決： フィールド名が正確かどうか確認してください。FormBridgeの設定画面で「フィールドコード」を確認できます。
問題： 郵便番号検索が失敗する
解決： ハイフンなしの7桁の郵便番号を入力してください（例：1000001）

❓ よくある質問

Q: 複数の機能を同時に使用できますか？

A: はい、複数の機能にチェックを入れることで、すべての機能を含んだコードが生成されます。

Q: 生成されたコードは安全ですか？

A: 公式のFormBridge APIを使用した安全なコードが生成されます。ただし、本番環境で使用する前には必ず動作確認を行ってください。

Q: カスタム機能の開発には時間がかかりますか？

A: カスタム機能は、要件によって開発時間が変わります。基本的な機能であれば、テンプレートとTODOコメントが生成され、簡単な修正で使用できます。
